# Lancer Speed Provider

A [Drag Ruler](https://foundryvtt.com/packages/drag-ruler) speed provider for lancer. It features integration with conditions to determine if a token can move and if so how many moves it can take. In the case of player mechs and npcs with the Limitless trait, it also shows a third range for units that can overcharge, indicating spending overcharge to boost.

## Installation

Paste the following manifest url into the foundry module installer:

```
manifest: https://github.com/BoltsJ/lancer-speed-provider/releases/latest/download/module.json
```
